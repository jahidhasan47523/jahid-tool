# JAHID TOOL v3.0

Facebook OTP Tool for Termux

## Features
- 🔐 Device-based activation with expiry
- 📱 FORGOT - Send OTP to numbers
- ✅ CONFIRM - Confirm OTP + Reset Password
- 🌐 PROXY - Proxy support
- ⚙️ SETTINGS - Customize threads, delay, password
